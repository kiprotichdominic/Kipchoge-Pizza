## Setup/Installation Requirements

- Open terminal (CTRL ALT T)
- Git clone https://github.com/kiprotichdominic/Delani-Studio.git
- cd DELANI-STUDIO
- On terminal type atom . to open using atom or type code . to open using Visual Studio.
